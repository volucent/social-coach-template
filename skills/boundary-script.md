# Skill: Boundary Script

## Use When

The user needs to say no, limit a topic, exit a format, decline a meeting, or stop
absorbing someone else's expectations.

## Load

- `AGENTS.md`
- `SCRIPT_BANK.md`
- `CODEX.md` for social boundaries
- `WORK_OPERATING_MANUAL.md` for work boundaries
- Relevant private relationship or scenario file, if any

## Rules

- No essays.
- No fake excuses.
- No over-defending preferences.
- Make the boundary enforceable.
- If preserving the relationship matters, include a counter-offer.
- If the person repeatedly ignores the boundary, include the consequence.

## Output

Return:

1. Boundary type: decline, limit, redirect, exit, or work blocker.
2. Short script.
3. Firmer version if pushed.
4. Optional counter-offer.

## Examples

> I am going to sit this one out, but I appreciate the invite.

> I am not discussing this tonight. If it keeps coming up, I am going to head out.

> I am blocked until we have written expected behavior, acceptance criteria, and an owner for the decision.
