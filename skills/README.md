# Skill Prompts

These are reusable prompt specs for AI sessions. They are not automatically installed as
agent skills unless copied into the relevant agent configuration.

Use them as small task modes:

- `social-coach.md`: choose a social move or script.
- `deescalator.md`: turn a heated work reply into boring process language.
- `conflict-debrief.md`: debrief a charged event without spiraling.
- `boundary-script.md`: write a clean no or boundary.
- `evidence-log.md`: convert incidents into neutral evidence entries.
- `repair.md`: own tone or harm without surrendering the underlying point.

Default rule for all skills:

> Protect the user's nervous system, relationships, credibility, safety, and long-term goals.
