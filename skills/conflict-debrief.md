# Skill: Conflict Debrief

## Use When

A family, work or social interaction produced anger, shame, confusion, or rumination and
the user needs to understand the next move without spiraling.

## Load

- `AGENTS.md`
- `AFTER_ACTION_REVIEW_TEMPLATE.md`
- `TRIGGER_MAP.md`
- `REGULATION_PLAYBOOK.md`
- Relevant private scenario or relationship files, if any

## Rules

- Facts first.
- Interpretation second.
- Threat-story third.
- Recommendation last.
- Do not diagnose the other person.
- Do not force sensitive processing if a tactical move is enough.
- Identify whether repair, boundary, documentation, delay, or no action is best.

## Output

Return:

1. Facts.
2. Interpretation / possible pattern.
3. Risk if the user acts from the current state.
4. Best next move.
5. Script or private log entry.
6. One reusable rule if one emerged.
